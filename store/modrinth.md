# Enchant Details for EMI

**Browse every enchantment in the game from the EMI screen — with all the details you actually want to know.**

Open the **Enchantment** category in [EMI](https://modrinth.com/mod/emi) (or search any enchanted book) and instantly inspect any enchantment: its levels, what it goes on, what it conflicts with, how rare it is, and where you can get it.

Works automatically with **modded enchantments** too — anything registered in the game shows up.

![In-game preview](https://raw.githubusercontent.com/Blushister/enchant-details-for-emi/main/store/screenshot.png)

## ✨ What you see for each enchantment

| Info | Detail |
|------|--------|
| 📖 **Levels** | Every level (1 → max) shown as an enchanted book |
| ⚔️ **Applicable items** | What the enchantment can be applied to |
| 🚫 **Conflicts** | Enchantments it can't be combined with |
| ⚖️ **Weight & cost** | Selection weight + per-level enchanting cost range |
| 🪧 **Availability** | Enchanting table · villager trades · loot · treasure · curse |
| 📝 **Description** | The enchantment's description, when provided |

## 📦 Requirements

- **NeoForge 1.21.1**
- **[EMI](https://modrinth.com/mod/emi)** — required, client-side

> ℹ️ This is a **client-side** mod. It does nothing on a dedicated server (safe to omit there).
